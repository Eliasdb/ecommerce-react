# Lendert's Garage

## About

An e-commerce store made for my friend to sell items he has in stock.

## Used tech

- React
- Axios
- Airtable
- Context API
- React Router Dom
- Netlify serverless functions
- Styled components

## Script to run locally

`npm i && npm run dev`
